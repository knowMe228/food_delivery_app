# 🍔 Food Delivery App (Demo)

Мобильное приложение доставки еды, написанное на **Python + KivyMD**.  
Все данные (рестораны, меню, отзывы) генерируются случайным образом и хранятся в локальной базе SQLite.  
Приложение работает полностью автономно (без серверов и API).

---

## 🚀 Установка и запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Заполните базу данных демо-данными:
   ```bash
   python data/seed_data.py
   ```

3. Запустите приложение:
   ```bash
   python main.py
   ```
